1. Boxplot.rmd 
   * Create a box plot function with various attributes such as:
     - Violin plot
     - Notch & Jitter
     - IQR 1, 2, 3
     - Mean, Min, Max, & Outlier (all with their values & the number of observations or IDs)
   * Additionally, I have created an additional theme:
     - Theme for ggplot (including transparency, axis, etc.)
     - Default title & labels (modifiable)
     - 3 default color palettes (modifiable or expandable as desired)
   * There are also examples on how to use it with sample data.
   * And many more features!
   * NOTE : 
     - However, there are 2 warnings in the boxp function that I'm not sure how to resolve.
     - I also have plans to add other features, such as:
       + Grouping by category (each category will have its own box plot with a different color)
       + Rotation and flipping options
       + Improvements to the violin plot shape
       + And many more, including handling large datasets.
